# 小红书内容整理台

一个手机可用的小工具，用来整理小红书收藏、点赞、分享链接，以及你复制下来的有用正文和评论。

它不会自动爬取小红书内容。你可以把自己觉得有用的帖子内容、评论、关键词粘贴进去，工具会自动分类、生成摘要、筛出要点，并保留原帖链接，方便以后搜索和跳转。

## 主要功能

- 粘贴小红书链接后手动补充标题、关键词、备注或评论
- 根据你填的关键词和备注自动细分类
- 自动生成核心摘要和要点
- 保留原帖链接，可一键打开
- 支持搜索标题、摘要、关键词、备注、链接
- 重复链接会提醒
- 详情页可以删除错误或不需要的条目
- 数据保存在当前浏览器本地
- 支持导出整理后的 JSON
- 可部署到 GitHub Pages 并添加到手机桌面

## 推荐使用流程

1. 在小红书复制帖子链接
2. 回到工具，点击“粘贴”
3. 点击“读剪贴板”，或手动粘贴链接
4. 点击“解析链接”
5. 自己补充：

```text
标题：给这条内容起一个好找的标题
关键词：比如 游戏推荐，悬疑电影，烹饪教程
备注 / 评论内容：复制你觉得有用的正文、评论、避雷点、推荐理由
```

6. 点击“加入整理台”

工具会根据这些内容生成分类、摘要和要点。

## 分类示例

工具会尽量按“领域 + 用途”生成细分类，例如：

```text
游戏·游戏推荐
游戏·游戏攻略
影视·片单推荐
影视·影评解析
美食·烹饪教程
美食·探店攻略
旅行·旅行攻略
学习·学习计划
绘画·灵感素材
阅读·清单推荐
```

## 删除和清理

如果链接粘错了、内容重复了，或者以后不需要了：

1. 在列表里点开那条内容
2. 进入详情页
3. 点击“删除这条”

新增时如果链接已经存在，工具会提醒你。

## 文件导入

也可以点击“导入”上传 `JSON / TXT / Markdown`。

TXT / Markdown 示例：

```text
Switch 必玩治愈游戏推荐
关键词：Switch，游戏推荐，治愈
评论里提到星露谷和动森很适合睡前玩，节奏慢，不需要强操作。
https://www.xiaohongshu.com/explore/xxxx

高分悬疑电影片单
关键词：电影，悬疑，片单
评论区推荐了几部反转强的片子，适合周末晚上看。
https://www.xiaohongshu.com/explore/yyyy
```

JSON 推荐字段：

```json
[
  {
    "title": "标题",
    "content": "正文或备注",
    "comment": "有用评论",
    "keywords": "游戏推荐，Switch，治愈",
    "source": "收藏",
    "link": "https://www.xiaohongshu.com/explore/xxxx"
  }
]
```

## GitHub Pages 部署

把这些文件放在仓库最外层：

```text
index.html
manifest.webmanifest
sw.js
icon.svg
README.md
```

然后在 GitHub 仓库里：

1. 打开 `Settings`
2. 进入 `Pages`
3. `Source` 选择 `Deploy from a branch`
4. `Branch` 选择 `main`
5. 文件夹选择 `/root`
6. 保存

当前项目新版链接：

```text
https://fjy934825-jpg.github.io/Little-Red-Book-Classifying-Tool/index.html?app=v9
```

## 手机添加到桌面

安卓 / 华为手机：

1. 用浏览器打开 GitHub Pages 链接
2. 确认页面能看到“导入、粘贴、示例、导出”
3. 打开浏览器菜单
4. 选择“添加到桌面”或“添加到主屏幕”

如果桌面版显示旧页面，删除旧图标，再用新版链接重新添加。

## 注意事项

- 工具不会自动登录或抓取小红书
- 数据保存在当前浏览器本地，换手机或换浏览器不会自动同步
- 建议定期点击“导出”备份 JSON
- 分类和摘要基于你粘贴的关键词、备注、正文和评论生成，内容越具体，效果越好
